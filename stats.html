📋 Supabase 建表指南
1. 打开 https://supabase.com 进入你的项目 2. 点击左侧 SQL Editor → New Query 3. 粘贴以下 SQL 并运行： CREATE TABLE page_views ( id BIGINT GENERATED ALWAYS AS IDENTITY PRIMARY KEY, page TEXT, title TEXT, referrer TEXT, screen TEXT, visited_at TIMESTAMPTZ DEFAULT NOW() ); ALTER TABLE page_views ENABLE ROW LEVEL SECURITY; CREATE POLICY "Anyone can insert" ON page_views FOR INSERT WITH CHECK (true); CREATE POLICY "Anyone can view" ON page_views FOR SELECT USING (true); 4. 运行成功后，刷新本页面就能看到云端数据了 🎉
