## 🚀 快速开始

1. **打开网站**：直接双击 `index.html`，用浏览器打开
2. **浏览页面**：顶部导航栏可以在 6 个页面之间切换
3. **修改内容**：用记事本/VSCode 打开任意 `.html` 文件，找到文字直接改

## 📖 学习路线（从零开始）

### 第一阶段：看懂结构（10分钟）

打开 `index.html`，对照以下概念看：

| 代码 | 是什么 | 类比 |
|------|--------|------|
| `<!DOCTYPE html>` | 告诉浏览器这是 HTML5 | 文件格式声明 |
| `<head>` | 放配置信息 | 身份证 |
| `<body>` | 放页面内容 | 你的身体 |
| `<header>` | 导航栏 | 网页的"头顶" |
| `<footer>` | 底部信息 | 网页的"脚" |

### 第二阶段：修改内容（20分钟）

找一段文字，比如在 `brands.html` 里找到：

```html
<p>Lululemon 是瑜伽裤行业当之无愧的"开山鼻祖"...</p>
```

把它改成你想写的话，保存，刷新浏览器 —— **你的第一个网页修改完成了** ✅

### 第三阶段：加新内容（30分钟）

**方法：** 复制一个已有的 `card` 代码块，粘贴，改里面的文字和颜色。

比如在首页加一个新品牌卡片：
```html
<div class="card">
    <div class="card-img" style="background: linear-gradient(135deg, #000, #444);">你的品牌</div>
    <div class="card-content">
        <span class="tag tag-serious">品牌百科</span>
        <h3>品牌名</h3>
        <p>你写的介绍文字</p>
        <a href="链接">查看 →</a>
    </div>
</div>
```

### 第四阶段：改样式（40分钟）

打开 `style.css`，在顶部的 `:root { }` 块里改颜色：

```css
--pink: #e94560;        /* 主色调，改成你喜欢的颜色 */
--gold: #f5a623;        /* 金色，改成其他配色 */
```

改完保存、刷新 —— 全站颜色都会变。

## ☁️ 部署上线（让全世界看到）

### 方法一：GitHub Pages（免费）

1. 注册 GitHub 账号 (github.com)
2. 新建仓库 → 上传这个文件夹里的所有文件
3. 进入仓库 Settings → Pages → 选 main 分支 → 保存
4. 等 2 分钟 → 你的网站在 `https://你的用户名.github.io` 上线

### 方法二：买域名（推荐）

1. 在阿里云/腾讯云/GoDaddy 买域名（比如 `yugaku.wiki`，约 ¥30~50/年）
2. 在 GitHub Pages 设置里绑定域名
3. 群友就能通过 `yugaku.wiki` 访问了

## 🛠️ 进阶学习推荐

有了这个基础之后，可以学：

1. **CSS Flexbox / Grid** → 理解页面布局怎么排的（本站用的就是 Grid）
2. **JavaScript** → 给本站加交互（比如真正的投票功能）
3. **Next.js / Vue** → 把静态站升级成动态站（有后台、数据库那种）
4. **SEO** → 让搜索引擎能搜到你的站
